# web

React 18 + TypeScript, Ant Design, TanStack Router (file-based) + TanStack Query, Chart.js.

## Key files

- **src/components/queries.ts**: All API query and mutation hooks (TanStack Query)
- **src/components/App.tsx**: Root app layout with router provider
- **src/routeTree.gen.ts**: Auto-generated by TanStack Router plugin (do not edit)

## Component layout

- **Configuration/**: CRUD forms and detail views for databases, syncables, ingestables, types
  - `Configurations.tsx` — shared list/layout component
  - `ConfigurationDetail.tsx` — shared detail view
  - Each sub-entity (Syncables/, Ingestables/, Databases/, Types/) has its own Form and optionally a Detail component
- **Proposals/**: Proposal list (`Proposals.tsx`) and add form (`AddProposalForm.tsx`)
- **Home/**: Dashboard with `Home.tsx`
- **Sider/**: Navigation sidebar
- **Types/**: Type-related components (under Configuration/Types/ for config, standalone for type views)

## Routes (file-based, src/routes/)

Routes map directly to URL paths. Each `*.tsx` file = a route. `*.lazy.tsx` = lazy-loaded.

- `/` — index (Home)
- `/databases`, `/databases/:databaseId`
- `/syncables`, `/syncables/:syncableId`
- `/ingestables`, `/ingestables/:ingestableId`
- `/types`, `/types/:typeId`

## API

The frontend calls `VITE_API_URL` (default `http://localhost:8080`, configured in `.env`).
